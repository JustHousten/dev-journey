# Übung 05 — Tabellen

## Erklärung

```html
<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Alter</th>
      <th>Stadt</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Justin</td>
      <td>25</td>
      <td>Berlin</td>
    </tr>
    <tr>
      <td>Anna</td>
      <td>30</td>
      <td>München</td>
    </tr>
  </tbody>
</table>
```

**Was bedeutet was:**
- `<table>` — die Tabelle selbst
- `<thead>` — Kopfbereich (Spaltenüberschriften)
- `<tbody>` — Inhaltsbereich
- `<tr>` — table row (eine Zeile)
- `<th>` — table header (Spaltenüberschrift, wird fett dargestellt)
- `<td>` — table data (eine Zelle)

## Aufgabe

Erstelle die Datei `05-tabellen.html`.

Baue zwei Tabellen:

1. Eine Tabelle mit deinen Lieblingsfilmen:
   - Spalten: Titel, Genre, Jahr, Bewertung (1-10)
   - Mindestens 5 Zeilen

2. Eine Tabelle mit einem Wochenplan:
   - Spalten: Montag bis Freitag
   - Zeilen: Morgen, Mittag, Abend
   - Füll ein paar Zellen mit Aktivitäten

## Ziel

- Du verstehst die Struktur einer HTML-Tabelle
- Du kannst `<thead>` und `<tbody>` sinnvoll einsetzen
